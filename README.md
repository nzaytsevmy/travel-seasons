# TravelTribe — Сезоны для путешествий

Сайт [traveltribe.ru](https://traveltribe.ru) — интерактивная таблица лучшего времени для 24 направлений с визовым гидом и калькулятором бюджета.

## Что внутри

- **index.html** — главная страница с динамическим разделом "Куда летят сейчас"
- **seasons.html** — таблица сезонов 24 × 12 месяцев
- **calculator.html** — калькулятор бюджета поездки из Москвы
- **fetch_prices.py** — скрипт обновления цен перелётов

## Технологии

Vanilla HTML + CSS + JavaScript. Без фреймворков, без сборщиков. Деплой через GitHub Pages.
