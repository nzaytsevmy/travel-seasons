# TravelTribe — Брендбук (Swiss / International Typographic)

**Статус: ОБЯЗАТЕЛЕН.** Все страницы и компоненты traveltribe.ru делаются строго по этому
документу. Любое отступление — только с явного разрешения Никиты. Источник истины — этот файл;
при конфликте с прочими дизайн-заметками приоритет здесь.

Язык дизайна: **International Typographic Style (Swiss)** — объективность, сетка, один гротеск,
иерархия размером/весом, только линейки, один акцент, много воздуха. Эталон — главная (вариант 1).
Анти-AI по построению: ноль эмодзи-декора, скруглений, теней, градиентов.

## 1. Цвет (палитра фиксирована)

| Токен | HEX | Роль | Контраст |
|---|---|---|---|
| `--paper` | `#f4f2eb` | фон (тёплый off-white) | — |
| `--ink` | `#1b1914` | основной текст, сильные линии | 14:1 на paper |
| `--mid` | `#6b685e` | вторичный текст, подписи, лейблы | 4.5:1 ✓ |
| `--line` | `#23211a` | сильные разделители (1px) | — |
| `--hair` | `#cdc9bd` | тонкие линейки между ячейками | — |
| `--gold` | `#8e6618` | ЕДИНСТВЕННЫЙ акцент (honeyed ochre) | 4.6:1 на paper ✓ AA |
| visa-required | `#56534c` | де-эмфаз статуса | 6:1 ✓ |
| row-hover | `rgba(142,102,24,.07)` | подсветка строки | — |

Правила: **один акцент** (gold) — ссылки, активные состояния, индексы, ключевое слово в заголовке,
«пик» в хитмэпе. Никаких вторых ярких цветов, никаких градиентов. `--gold` — это **пол по контрасту**:
светлее охру не делать (упадёт ниже AA 4.5:1). Тёмная тема — при необходимости инверсия тех же ролей.

## 2. Типографика

- **Один гротеск: Golos Text** (`'Golos Text Variable'`, self-hosted via fontsource; Helvetica/Arial fallback).
  Никаких вторых семейств, никакого серифа, никакого Inter/Roboto/Space Grotesk.
- **⚠️ Шрифт ОБЯЗАН поддерживать кириллицу** (основной язык сайта — русский). Golos Text — кириллица-натив.
  НЕ брать Archivo/латиница-only (был баг: кириллица падала в системный fallback). Проверять отрисованный
  глиф, а не `computed font-family`.
- Иерархия — только **размером и весом** (400/500/600/700/800/900), не гарнитурой.
- **Табличные цифры везде:** `font-variant-numeric: tabular-nums lining-nums`.
- Выравнивание **flush-left, ragged-right**. Центрирование запрещено (кроме мелких служебных).
- Display (H1): `clamp(2.4rem,7vw,6.4rem)`, вес 800, `line-height:.94`, `letter-spacing:-.035em`.
- Заголовок секции (H2): `clamp(1.4rem,2.6vw,2rem)`, вес 800, `-.02em`.
- Тело: `1–1.18rem`; данные/таблица `.92rem`.
- Функциональные микро-лейблы (хедеры таблиц, подписи фильтров/форм, verdict-теги статуса): `.64–.74rem`,
  вес 700, UPPERCASE, `letter-spacing .04–.06em`, цвет `--mid`. **Только внутри таблиц/контролов.**
- **⛔ Eyebrow-капсы над заголовками ЗАПРЕЩЕНЫ** (решение Никиты 2026-07-06): «РАЗДЕЛ · 2026» капсом
  с трекингом над H1 — узнаваемый признак сгенерированного дизайна. Вместо них — обычная строка
  `.8–.86rem`, вес 600, sentence case (gold или `--mid`), либо ничего.
- **⛔ Декоративные золотые индексы секций/карточек («01», «02»…) ЗАПРЕЩЕНЫ** (то же решение).
  Номер допустим только когда порядок семантичен для читателя (шаги маршрута «день 1/2/3»).

## 3. Сетка и пространство

- **12 колонок**, `max-width:1280px`, gutter `clamp(.8rem,2vw,1.6rem)`, поля `clamp(1.1rem,3vw,2.6rem)`.
- **Асимметрия** обязательна (Swiss): сильная левая колонка, мета/служебное — в угол; не симметричные «по центру» блоки.
- Mobile <760px: грид схлопывается в 1–2 кол; **таблицы → строки-блоки** с подписями `::before`, не горизонтальный скролл-клон.
- Дыхание щедрое; плотность — за счёт линеек, не рамок.

## 4. Поверхности и разделители

- **ТОЛЬКО hairlines.** `border-top:1px solid var(--line)` (сильная) / `var(--hair)` (тонкая между ячейками).
- **ЗАПРЕЩЕНО:** `border-radius` (всё прямоугольное, 0), `box-shadow`, любые тени, градиенты, заливки-карточки.
- Секции разделяются `border-top:2px solid var(--line)` + H2 (без индекс-цифры — см. §2).

## 5. Компоненты (канон)

- **Дата-таблица:** top/bottom hairline, шапка — мелкий uppercase `--mid`, табличные цифры, flush-left,
  hover-tint строки, на мобайле — блоки с `data-label`. Первая колонка — заподлицо с сеткой, у остальных — левый отступ от линейки.
- **Фильтры:** текст-тогглы (НЕ пилюли). Активный = `--gold` + подчёркивание, приклеенное к тексту
  (`text-decoration` + `text-underline-offset`, не border-bottom). Хит-зона ≥44px.
- **Ссылки/CTA (канон 2026-07):** текст `--gold`, вес 600–700, тонкое подчёркивание
  `text-decoration: underline 1px color-mix(in srgb, var(--gold) 55%, transparent)`,
  `text-underline-offset:.22em`, `text-decoration-skip-ink:auto`; hover — полный gold underline.
  НЕ border-bottom (рвётся при переносе), НЕ 2px-линия, sentence case (не UPPERCASE), без `white-space:nowrap`
  на многословных ссылках. Хит ≥44px.
- **⛔ Стрелка «→» как декор ссылки/кнопки ЗАПРЕЩЕНА** (решение Никиты 2026-07-06): «Текст →» в каждом CTA —
  главный «сделано ИИ»-маркер. Стрелка допустима только как семантика: маршрут («Москва → Пхукет» в проозе
  и заголовках рейсов), prev/next-пагинация месяцев («← июнь · август →»), путь в настройках.
- **Перечни (value-list):** hairline-грид, первая ячейка без левого отступа (без золотых индексов).
- **Кнопок-пилюль нет.** Действие по умолчанию = типографика + тонкое подчёркивание. Микрокопию
  CTA разнообразить по контексту («Читать дальше», «Открыть месяц», «Найти билет»), не штамповать одну форму.
- **Заливная primary-CTA — разрешённое исключение** (решение Никиты 2026-07-02): ОДНА на
  страницу/секцию, только главное партнёрское действие. Прямоугольная (0 скруглений/теней),
  фон `--gold`, текст `#fff` (контраст 18.7:1), высота ≥48px, benefit-текст + цифра (без стрелки).
  Глобальный класс `.aff-btn`. Все вторичные действия остаются hairline-ссылками.

## 6. Иконки и изображения

- **Иконок нет.** Эмодзи как декор — ЗАПРЕЩЕНЫ. Если иконка реально нужна — один монохромный line-набор (lucide), не emoji, не разностиль.
- Фото: `object-fit:cover`, лёгкая десатурация (`filter:grayscale(.06) contrast(1.02)`; портрет автора `grayscale(.3)`). Только webp, никогда avif. width/height для CLS=0.

## 7. Движение

- Сдержанно: переходы `.15–.6s` на цвет/подчёркивание/`transform`. `scroll-behavior:smooth`.
- Без каруселей, без авто-анимаций, без параллакса-ради-параллакса. `prefers-reduced-motion` уважать.

## 8. Доступность (часть брендбука, не опция)

- Контраст текста **≥4.5:1** (gold `#8e6618` — нижняя граница акцента).
- Тач-таргеты главных контролов **≥44px**, минимум 24px везде.
- `:focus-visible{outline:2px solid var(--gold);outline-offset:3px}` на всём интерактиве.
- `lang="ru"`, alt на смысловых картинках, meta-description, один H1, иерархия H2/H3.
- Цель: Lighthouse A11y = 100.

## 9. Контент и тон

- Польза на виду, без копания: реальные данные инлайн (₽, виза, сезон), а не ссылки-обещания.
- От первого лица, конкретика, «был лично», ноль воды и маркетинг-штампов. Русский.
- Монетизация по [[feedback_monetization_ctr]]: benefit-CTA (выгода+цифра, без стрелки), deep-links,
  `rel="sponsored noopener"` + новая вкладка + erid в URL где партнёр даёт (38-ФЗ). БЕЗ видимой
  надписи «реклама» (решение Никиты 2026-06); мягкий disclosure — «партнёрская ссылка, цена не меняется».

## 10. Готовые токены (вставлять как есть)

```css
:root{
  --paper:#f4f2eb; --ink:#1b1914; --mid:#6b685e; --line:#23211a; --hair:#cdc9bd; --gold:#8e6618;
  --ff:'Golos Text Variable','Golos Text',Helvetica,Arial,sans-serif;
  --container:1280px; --gutter:clamp(.8rem,2vw,1.6rem); --edge:clamp(1.1rem,3vw,2.6rem);
}
body{font-family:var(--ff);font-variant-numeric:tabular-nums lining-nums;background:var(--paper);color:var(--ink)}
:focus-visible{outline:2px solid var(--gold);outline-offset:3px}
```
